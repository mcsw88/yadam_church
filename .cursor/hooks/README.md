# Cursor Hooks (yedam)

이 디렉터리는 **예시만** 둔다. **실제 자동 실행 연결은 하지 않는다.**

## 현재 상태

- `hooks.json` 없음 — 생성하지 않았다
- Cursor가 hook을 실행하도록 **설정하지 않음**

## 예시 파일

- `examples/block-dangerous-command.example.js` — 위험한 셸 패턴 감지 시 경고하는 **샘플**
- `examples/after-edit-check.example.js` — 편집 후 사용자에게 `lint` / `build` 확인을 안내하는 **샘플**

프로젝트에 맞게 복사·수정한 뒤, **별도 승인**을 받고 `hooks.json` 등을 연결한다.
