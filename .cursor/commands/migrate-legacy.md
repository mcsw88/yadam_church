# migrate-legacy

루트 레거시 HTML/CSS/JS/`server.js`는 **사용자가 명시적으로 허용한 경우에만** 다룬다.

## 사용 시 입력

- 어떤 레거시 파일을 건드릴 수 있는지 명시
- 목표 (참고만 / Next로 이전 / 삭제는 별도 결정 등)

## 에이전트 지시

1. 허용 범위가 없으면 **중단**하고 기본 정책(레거시 수정 금지)을 알린다
2. 허용된 경우에만 매핑 표(레거시 경로 → `app/` 등) 초안을 제시한다
3. `docs/migration/` 기존 문서와 충돌 여부를 확인한다
4. 코드 변경은 사용자 승인 후
5. `.cursor/skills/legacy-migration/SKILL.md` 참고
