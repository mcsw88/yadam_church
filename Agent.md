# AGENTS.md

## 프로젝트 목적
이 프로젝트는 전통적인 한국 교회 웹사이트 톤을 유지하면서, 안정적인 반응형 UI를 구현하는 것을 목표로 한다.

## 절대 금지
- 스타트업 랜딩페이지 느낌 금지
- 과한 카드 UI 금지
- 과한 여백 금지
- 전체 페이지를 한 번에 대규모 수정 금지
- 요청받지 않은 색상 체계 변경 금지

## 작업 원칙
- 작업 전 현재 구조를 먼저 분석한다
- 수정 계획을 먼저 제시한다
- 한 번에 한 섹션만 수정한다
- 불확실하면 추정하지 말고 확인한다
- 완료 후 변경 파일과 이유를 요약한다
- 코드는 500줄이하로 한다

## 반응형 원칙
- 고정폭 금지
- px 남발 금지
- 모바일 메뉴 겹침 금지

## 검증 원칙
- 가능한 경우 미리보기로 데스크톱/모바일 둘 다 확인한다
- 승인 기준을 만족하지 못하면 완료로 간주하지 않는다
