# PROPOSAL_INBOX.md

들어온 제안·아이디어·외부 피드백을 **코드에 바로 넣지 않고** 여기에 먼저 정리한다.  
반영 전에는 분류·승인 절차를 거친다 (`06-proposal-integration.mdc`, `integrate-proposal` command).

## 원칙

- 제안 → 이 문서에 기록 → 반영 위치 분류 (문서 / rule / skill / task / 코드) → **승인 후** 각 레이어에 반영
- 코드 반영은 별도 작업·승인에서 수행

## 템플릿 (복사 후 사용)

```text
### [PROP-번호] 제목

- **출처:** (채팅, 이슈, 본인 등)
- **요약:** (한글)
- **분류 (예정):** docs | rule | skill | task | code
- **상태:** inbox | triaged | approved | rejected | done
- **비고:**
```
